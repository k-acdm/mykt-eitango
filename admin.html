💎 LINE通知管理

各生徒の 保護者LINE / 生徒LINE の登録状況と、通知の ON/OFF を確認・変更できます。
通知の状態：未設定（学齢で自動判定：小中=ON / 高=OFF）／ ON（強制配信）／ OFF（強制停止）。
通知欄をクリックすると 未設定 → ON → OFF → 未設定 の順で切り替わり、その場で保存されます。
⏳ 読み込み中...

🎯 絶対ミッション設定

各生徒の 絶対ミッション（両輪システムで「やったら得 + やらないと損」となるコンテンツ）を切り替えます。
英単語RUSH / 三語短文：学齢に関係なく ON/OFF 可（高校生も opt-in 可能）。
和文英訳① / カンジー：小・中学生のみ ON 可（学齢設定が必要）。
基礎計算：中学生のみ ON 可（教科内容上の妥当性ガード）。
トグルで TRUE / FALSE を切替（その場で保存、空欄は FALSE 扱い）。
